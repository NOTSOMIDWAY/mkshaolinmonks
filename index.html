⚔ Latest Discoveries ⚔
2025-05-01
GAMEDATA.WAI Fully Extracted — 103 ADX Audio Files NEW
First complete extraction of the CRI AFS audio archive. All 103 tracks decoded from CRI ADX format to WAV. Filenames recovered from AFS filename table. 22.7 minutes of audio including cutscene music, voice lines, and ambient SFX.
2025-05-01
Engine Name Confirmed: Paradox Engine NEW
Debug symbols in SLUS_210.87 reveal the full engine source tree under /Projects/Paradox/. Internal project codename confirmed as "Mka" (Mortal Kombat: Adventure). Havok Physics, CRI Middleware, and Flash/Scaleform UI all identified.
2025-05-01
PWF Archive Format Documented
GAMEDATA.WAD header decoded. Magic bytes PWF, 7,139 entries, 2048-byte sector alignment. EWDF and ps2_DTPK sub-formats identified within the archive body. WADCRC.BIN confirmed as flat CRC32 table — one entry per WAD asset.
2025-05-01
Hidden 9th WAD File: GAMEDATA.WAK
ELF executable references GAMEDATA.WAK which does not exist on the retail disc. Cut content or a different regional version. Investigation ongoing.
